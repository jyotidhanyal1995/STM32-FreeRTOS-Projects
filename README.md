# STM32-FreeRTOS-Projects

FreeRTOS projects implemented on STM32F103C8T6 (Blue Pill) using STM32CubeIDE.

## Projects

### 1. Free_RTOS_Task
- Dual-task LED control using FreeRTOS task scheduling
- Demonstrates task creation, priority, and independent execution

### 2. Free_RTOS_Queue
- Producer-consumer communication between tasks using FreeRTOS Queue
- Demonstrates inter-task data passing

### 3. FreeRTOS_Semaphore
- Binary semaphore signaling between tasks
- Demonstrates task synchronization

### 4. FreeRTOS_Mutex
- Shared resource protection using FreeRTOS Mutex
- Demonstrates mutual exclusion to avoid resource conflicts

## Hardware
- Board: STM32F103C8T6 (Blue Pill)
- IDE: STM32CubeIDE
- Debugger: ST-Link v2
